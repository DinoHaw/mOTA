# stm32f10x 裸机 Demo

---

## 1、简介

通过 `app\src\app.c` 的 `void sfud_demo(uint32_t addr, size_t size, uint8_t *data)` 方法来演示并测试第一个 Flash 设备的擦除、写入及读取功能，最后会将写入后数据读取出来，与写入前的数据进行对比，如果一致，则测试通过。

### 1.1、使用方法

- 1、打开电脑的终端与Demo的串口1进行连接，串口配置 115200 8 1 N
